## first steps

- mkdir api-express-ts
- cd api-express-ts
- npm init -y
- npm install typescript -D
- code .

---

## init project

- npm run tsc -- --init
- > edit if needed the tsconfig.json file for different options
- npm run tsc // add script "tsc" un package.json
